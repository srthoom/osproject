1. Diners are allocated tables on first come first served (FCFS) basis.

2. Activity log is generated for the entire duration, the restaurant is open.

3. Diners and Cooks are implemented as JAVA threads and Tables and Machines are treated as resources.

4. All diners that arrived within 120 mins are served.

5. The restaurnat remains open till 120 mins or till the last diner finishes eating, whichever occurs later.
